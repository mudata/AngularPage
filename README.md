# AngularPage
AngularPage with authentication Login and Register and Admin privilege
Database with nodeJs and MongoDb with mongoose
# Functionality
Add Item
Edit Item
Delete Item
Add to CartItem
Search bar
Sort
Profile Page
Contact Form
Submit Cart Form
Add comments on Items
About Page with Angular Animation
Header and Footer Component
# Install packages
npm i
# Start Server 
node server
# Run the application
ng s
